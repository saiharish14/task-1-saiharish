# BrainFlow AI Landing Page

A modern and responsive AI SaaS landing page built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

🔗 https://saiharish14.github.io/task-1-saiharish/

## 📂 GitHub Repository

🔗 https://github.com/saiharish14/task-1-saiharish

## 📌 Features

* Modern AI-inspired UI
* Fully responsive design
* Smooth scrolling navigation
* Glassmorphism cards
* Interactive pricing section
* Testimonials section
* Contact form with popup feedback
* Hover animations and transitions
* Mobile-friendly layout
* Built with pure HTML, CSS, and JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
